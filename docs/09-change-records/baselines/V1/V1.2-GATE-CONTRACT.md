# V1.2 Gate Contract

Status: Closed for V1.2

Version: V1.2-GATE-CONTRACT

Date: 2026-06-07

## Purpose

Define the V1.2 gate sequence for the SuperSpec schema baseline iteration.

## V1.2 Scope

V1.2 covers:

- SuperSpec schema design baseline
- SuperSpec activation contract
- SuperSpec artifact map
- SuperSpec static schema assets under `openspec/schemas/superspec/`
- formalized SuperSpec usage guidance

V1.2 does not cover:

- business/frontend implementation
- full runtime validation execution
- Superpower mission or power outputs

## Required Gates

1. Design review
2. Design disposition
3. Design closure verification
4. Implementation review
5. Implementation disposition
6. Implementation closure verification
7. Acceptance precheck
8. Acceptance precheck verification

## Exit Condition

V1.2 is ready for user acceptance review when:

- all V1.2 outputs exist under `baselines/V1/` or their declared doc targets
- the declared `openspec/schemas/superspec/` assets exist
- `docs/03-openspec/superspec-usage.md` reflects the V1.2 baseline
- no V1.2 gate record is blocked
- no activation claim contradicts the non-business-code constraint

User acceptance is required before V1.3 may start.
