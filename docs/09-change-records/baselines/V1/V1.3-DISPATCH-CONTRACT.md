# V1.3 Dispatch Contract

Status: Closed for V1.3

Version: V1.3-DISPATCH-CONTRACT

Date: 2026-06-07

## Purpose

Define the bounded dispatch format for any agent assigned to V1.3 Superpower governance work.

## Required Dispatch Fields

- target branch
- target minor version
- target stage
- target mission
- allowed files
- forbidden files
- required reads
- validation commands
- evidence output path
- final report format

## V1.3-Specific Boundaries

Agents working in V1.3 must not:

- change `src/`
- change `tests/`
- change runtime/business implementation files
- claim end-to-end integration proof
- claim V1 major-version completion

## Required Output Types

- Superpower governance documents
- baseline design documents
- review/disposition/closure records
- acceptance-precheck records

## Acceptance Rule

Any V1.3 output that implies unrestricted agent authority, runtime validation that was not executed, or user acceptance that was not given must be rejected.
