## Scope

V1 is the first version that must define:

- how change management is controlled
- how design flow is gated
- how brainstorm and agent execution are governed
- how later code-bearing validation work is allowed to enter implementation

V1 is not closed until these rules are usable by execution agents and tied to verification and release records.
