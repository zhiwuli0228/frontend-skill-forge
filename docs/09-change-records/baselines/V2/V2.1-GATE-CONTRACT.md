# V2.1 Gate Contract

Status: Draft baseline

Version: V2.1-GATE-CONTRACT

Date: 2026-06-07

## Purpose

Define the V2.1 gate sequence for version design and runtime-surface architecture.

## V2.1 Scope

V2.1 covers:

- V2 intent and scope definition
- runtime-surface design for dashboard, task list, and task creation
- minor-version sequencing for the first code-bearing version
- initial verification expectations for V2

V2.1 does not cover:

- runtime implementation of the pages themselves
- final V2 acceptance
- backend integration

## Required Gates

1. IR review
2. IR disposition
3. IR closure verification
4. SR review
5. SR disposition
6. SR closure verification
7. design review
8. design disposition
9. design closure verification
10. acceptance precheck
11. acceptance precheck verification

## Exit Condition

V2.1 is ready for user acceptance review when:

- V2 IR, SR, design baseline, and iteration plan exist
- the three target runtime surfaces are explicitly scoped
- sequencing into code-bearing iterations is explicit
- no V2.1 artifact claims that runtime implementation has already happened
