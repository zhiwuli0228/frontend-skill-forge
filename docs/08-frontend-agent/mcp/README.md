# MCP Exploration Templates

Status: Draft

## Purpose

Provide reusable templates for MCP exploration sessions.

## Scope

All MCP exploration activities using Playwright.

## Contents

- `exploration-session-template.md` - Session record template
- `page-observation-template.md` - Page observation template
- `interaction-trace-template.md` - Interaction trace template
- `selector-stability-guide.md` - Selector stability assessment guide

## Usage

Copy and customize templates for each exploration session.
Fill in all required fields.
Store completed records in `docs/07-evidence/` or project-specific evidence directories.
