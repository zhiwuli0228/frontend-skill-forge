# Schema Templates

Status: Draft

## Purpose

Provide lightweight JSON schema templates for data structures.

## Scope

All data structures in the frontend agent layer.

## Contents

- `mcp-session.schema.json` - MCP session schema
- `ui-evidence.schema.json` - UI evidence schema
- `e2e-asset.schema.json` - E2E asset schema
- `skill-evolution.schema.json` - Skill evolution schema

## Usage

Use schemas to validate data structures.
Do not over-engineer - keep schemas lightweight.
Update schemas when data structures change.
