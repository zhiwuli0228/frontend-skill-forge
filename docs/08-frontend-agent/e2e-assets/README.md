# E2E Asset Templates

Status: Draft

## Purpose

Provide templates for E2E test asset management.

## Scope

All E2E test assets in the frontend agent layer.

## Contents

- `e2e-case-registry-template.md` - E2E case registry template
- `smoke-test-contract.md` - Smoke test contract template
- `regression-test-contract.md` - Regression test contract template
- `test-data-policy.md` - Test data management policy

## Usage

Use templates when creating or managing E2E test assets.
Maintain registry for all E2E cases.
Follow lifecycle defined in `e2e-asset-lifecycle.md`.
