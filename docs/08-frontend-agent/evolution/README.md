# Evolution Templates

Status: Draft

## Purpose

Provide templates for Skill, Rule, and Knowledge evolution.

## Scope

All evolution activities in the frontend agent layer.

## Contents

- `skill-evolution-queue.md` - Skill evolution queue
- `rule-evolution-queue.md` - Rule evolution queue
- `knowledge-evolution-queue.md` - Knowledge evolution queue
- `evidence-to-skill-promotion.md` - Promotion process
- `human-feedback-log.md` - Human feedback log
- `evolution-decision-record-template.md` - Decision record template

## Usage

Track evolution candidates in queues.
Record human feedback before promotion.
Document all evolution decisions.
