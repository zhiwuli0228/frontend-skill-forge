## Process

1. Evidence triggers candidate
2. Candidate added to queue
3. Validate accuracy
4. Verification before acceptance
5. Knowledge updated or rejected

## Rules

- Knowledge must be factually accurate
- Cite sources when possible
- Archive outdated knowledge
- Track all decisions
