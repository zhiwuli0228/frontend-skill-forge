# UI Evidence Templates

Status: Draft

## Purpose

Provide reusable templates for UI evidence collection.

## Scope

All UI evidence artifacts from MCP exploration and testing.

## Contents

- `evidence-capture-template.md` - Evidence capture template
- `evidence-index-template.md` - Evidence index template
- `route-discovery-template.md` - Route discovery template
- `component-discovery-template.md` - Component discovery template
- `ui-state-snapshot-template.md` - UI state snapshot template
- `failure-diagnosis-template.md` - Failure diagnosis template

## Usage

Use templates when capturing evidence.
Fill in all required metadata.
Store completed evidence in project-specific evidence directories.
